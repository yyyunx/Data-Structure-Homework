# HW4 - Recursive Knight's Tour

Solve the Knight's Tour problem using recursion and backtracking.

### Features

- Start from the upper-left corner of the chessboard
- Visit every square exactly once
- Use recursive backtracking to search for a valid path
- Print the visiting order if a solution exists
- Print `no solution` if no valid path exists

### Techniques

- Recursion
- Backtracking
- DFS
- Knight Move Simulation

### Note

This homework solves the same Knight's Tour problem as HW3, but uses recursive backtracking instead of a self-defined stack.
HW3 uses an iterative approach with a custom stack, while HW4 uses function recursion to manage the backtracking process.
