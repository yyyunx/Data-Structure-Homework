# HW3 - Knight's Tour

Solve the Knight's Tour problem using a self-defined stack instead of recursion.

### Features

- Start from the upper-left corner of the chessboard
- Visit every square exactly once
- Use backtracking to search for a valid path
- Print the visiting order if a solution exists
- Print `no solution` if no valid path exists

### Techniques

- Self-defined Stack
- Backtracking
- Iterative DFS
- Knight Move Simulation
