# HW8 - Huffman Compression

Implement Huffman compression and decompression with a simple Windows GUI.

### Features

- Huffman Encoding
- File Compression
- File Decompression
- Binary File Processing
- Windows GUI for File Selection

### Techniques

- Huffman Tree
- Priority Queue
- Binary File I/O
- Tree Serialization
- C++/CLI Windows Forms
